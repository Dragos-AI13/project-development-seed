# Chat Closeout Prompt

Use this prompt before ending a coding session:

---
Before closing this session:
1) Update `docs/state/PROJECT_STATUS.md`.
2) Update `docs/state/NEXT_ACTIONS.md` with strict order.
3) Update `docs/state/CURRENT_SESSION.md`.
4) Update `docs/state/CHANGELOG_WORKING.md`.
5) Update `docs/state/DECISION_LOG.md` if decisions were made.
6) If architecture changed, update ADRs under `DECISIONS/`.
7) Provide a short closeout summary.
---
