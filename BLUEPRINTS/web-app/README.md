# Blueprint: Web App

## Suggested Stack

- Frontend: React + TypeScript
- Backend: Node.js/FastAPI
- DB: PostgreSQL

## Core Modules

- `auth`
- `users`
- `core-ui`
- `features/*`
- `api-client`

## Mandatory Gates

- Input validation on client and server
- AuthZ checks on API side
- Unit + integration tests on critical flows
