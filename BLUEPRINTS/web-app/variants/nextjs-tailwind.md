# Variant: Next.js + TypeScript + Tailwind

## Choose When

- You need SEO, SSR, or hybrid rendering.
- You want fullstack web velocity with one framework.

## Avoid When

- Product is purely internal SPA without SSR needs.
- You need minimal runtime complexity.

## Trade-offs

- Pros: strong ecosystem, flexible rendering model.
- Cons: higher complexity around server/client boundaries.

## Baseline Stack

- Next.js
- TypeScript
- Tailwind CSS
- React Testing Library + Playwright
